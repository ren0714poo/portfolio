#webサイト

松岡 蓮・ポートフォリオサイト

概要
友人たちの投稿した飲食店がマップに表示され、
みんなのおすすめの飲食店が見れる、SNS要素の強いマップアプリです。
制作期間
約2ヶ月
担当範囲
企画 / デザイン / コーディング / レスポンシブ対応
使用ツール
Figma / Illustrator / HTML / CSS / JavaScript

制作目的

飲食店を見つけれるアプリがあれば便利だなと思っていたのと、
APIを起用した制作をしたこたがなかったのでGoogleMapのAPIを起用し、
今回の制作をしました。

デザイン

フォローした方の投稿が写真でマップに表示されるので、
最初にマップを開いたときに料理が見れるので、最初にどんなのがあるかっていうのが
瞬時にわかるようにデザインしました。

ターゲット

なかなか外食先を決めれない人、SNSが好きな人
