#webサイト

松岡 蓮・ポートフォリオサイト

概要
Webデザイナーとしての自己紹介や制作実績をまとめたポートフォリオサイトです。
制作期間
約3週間
担当範囲
デザイン / コーディング / レスポンシブ対応
使用ツール
Figma / Illustrator / HTML / CSS

制作目的

自分のスキルや世界観を表現し、採用担当者が見やすく印象に残るデザインを目指しました。

デザイン

「見る人に親しみを感じてもらえるデザイン」をテーマに制作しました。
暖色系のカラーを中心に構成することで、温かみや人とのつながりをイメージさせています。
また、写真やボタンを角丸にすることで、全体に柔らかさと安心感を演出しています。
